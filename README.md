# Crime Analysis Dashboard - India 🇮🇳

This Power BI dashboard analyzes crime trends across Indian states.

## 📊 Features
- Total crimes KPI
- State-wise crime distribution
- Crime rate per 100k population
- Top 5 crime-prone states
- Crime trend (2020–2022)
- Interactive map

## 🛠 Tools Used
- Power BI
- DAX
- Data Visualization
